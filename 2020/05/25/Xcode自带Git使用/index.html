Xcode自带Git使用

发表于 2020-05-25 更新于 2021-04-06 分类于 Apple开发

本文主要介绍xcode自带的Git功能的使用。为什么要学xcode自带的Git管理工具呢？因为它足够简单、足够方便、足够好用。

  • 简单：只需简单几步就可以生成本地Git仓库

  • 方便：xcode自带的工具，写完一个功能直接提交，方便性不言而喻

  • 好用：提交时可以清楚的看到自己改动了那些东西，改前改后是什么样的，左右对照，清楚明了。

Tip1:新建项目时同时创建本地Git仓库


如图所示：在创建 Xcode 工程时，会自动勾选“在 Mac创建 Git 资源”

小贴士：在Mac电脑下使用command + shift + .可以切换隐藏文件的显示与隐藏。我们可以发现，工程文件下有个.git文件夹，即为我们的 Git 资源文件。
